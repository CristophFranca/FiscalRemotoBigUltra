## 📖 Sobre Esta Documentação

Esta documentação foi criada para centralizar todos os procedimentos, funções e boas práticas relacionadas às operações de **Fiscalização de Caixa** e **Prevenção de Perdas** da rede BigUltra.

O material está dividido em módulos específicos para facilitar a consulta rápida e o treinamento de equipes.

<br>

---

## 🗂️ Estrutura da Documentação

### 👥 [**FISCAL.md**](FISCAL.md) - Fiscais de Caixa Remoto

Documentação completa para fiscais de caixa, incluindo:

- ⌨️ **Atalhos do Teclado** - Comandos rápidos para navegação e operações
- 🛒 **Funções do Sistema** - Todas as funções numeradas organizadas por categoria
- 💳 **Procedimento POS** - Informações de transações e dados importantes
- 📝 **Procedimentos** - Passo a passo de operações comuns (TEF, devoluções, descontos)
- 🚨 **Erros Comuns** - Problemas recorrentes e como resolvê-los

**[📄 Acessar Documentação Completa →](FISCAL.md)**

<br>

### 🛡️ [**PREVENCAO.md**](PREVENCAO.md) - Prevenção de Perdas

Guia de erros operacionais que causam prejuízos e como evitá-los:

- 🔢 **Erro de Multiplicador** - Quando quantidades são lançadas incorretamente
- 🍺 **Cervejas** - Atenção especial às variações de pacotes (6, 8, 12, 15 und)
- 🍫 **Achocolatados** - Diferenças entre Italakinho (24) e Quatorzinho (27)
- 🧃 **Produtos com Sabores** - Impacto no estoque ao registrar sabor errado
- 📋 **Checklist de Prevenção** - Verificações antes de finalizar vendas

**[🛡️ Acessar Guia de Prevenção →](PREVENCAO.md)**

## 📊 Resumo Rápido

### Atalhos Mais Usados

| Tecla | Função |
|:-----:|--------|
| **F4** | Estorno |
| **U** | Menu de Funções |
| **T** | Totalizar Cupom |
| **C** | Consulta Preço |
| **Z** | Cancelar Cupom |

### Funções Mais Comuns

| Código | Descrição |
|:------:|-----------|
| **Função 13** | Cancelamento de TEF |
| **Função 17** | Estorno/Impressão |
| **Função 29** | Divergência de Preço |
| **Função 42** | Devolução de Produto |
| **Função 44** | Devolução de ValeCredito |

### Erros Mais Frequentes

| Erro | Causa | Solução |
|------|-------|---------|
| **Multiplicador** | Digitação incorreta de quantidade | Conferir visualmente antes de finalizar |
| **Pesagem** | Sistema trava em 1kg | Calcular manualmente e usar Função 29 |
| **Sabores** | Todos registrados como mesmo sabor | Bipar cada sabor separadamente |
| **Conexão Vale** | Servidor offline | Liberar em dinheiro + Função 44 depois |
